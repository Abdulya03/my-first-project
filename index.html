Information about the programmer



Барномасозӣ чист?


Барномасозӣ (ё программирование) раванде мебошад, ки дар он инсон (барномасоз) ба компютер фармон медиҳад, ки чӣ кор кунад. Ин фармонҳо бо истифодаи забонҳои барномасозӣ навишта мешаванд , мисли Python, JavaScript, C++, ва ғайра.

Барномасоз чӣ кор мекунад?


Барномасоз: Барномаҳо месозад (мисли бозиҳо, вебсайтҳо, мобил-аппҳо, системаҳои автоматиконидашуда); Код менависад, месанҷад ва хатогиҳоро ислоҳ мекунад; Бо дастаҳо (дизайнер, тестер, мизоҷ) кор мекунад; Дар бораи алгоритмҳо ва маълумот хуб медонад.

Чаро барномасозӣ муҳим аст?


Зеро: Ҳама чиз дар ҷаҳони муосир бо барномаҳо кор мекунад: телефонҳо, бонкҳо, дронҳо, интернет; Барои инсон кор осон ва автоматӣ мешавад; Даромади баланд ва имконияти кор аз фосилаи дур дорад.

Frontend


🔹 1. Фронтенд (Frontend) — “Қисми намоён” Фронтенд — он чизест, ки корбар мебинад ва бо он кор мекунад:

📱 Дар мисоли вебсайт:
Тугмаҳо
Меню
Расмҳо
Формҳо
Дизайн ва аниматсия

📦 Забонҳои фронтенд:
HTML — сохтори саҳифа
CSS — зебо кардани саҳифа (ранг, ҷойгиршавӣ, эффектҳо)
JavaScript — зинда кардани саҳифа (мисли клик, ҳаракат, анимация)
🛠️ Framework-ҳо:
React.js
Vue.js
Angular

Backend


🔹 2. Бекенд (Backend) — “Қисми пушти саҳна” Бекенд — он қисм аст, ки дар паси парда кор мекунад, мисли:

Кор бо базаи маълумот (users, posts, comments…)
Кори логикаи бизнес (мисли ҳисоб кардан, тасдиқ кардан)
Иҷозатнома (login, signup)
Пайваст шудан ба серверҳо

📦 Забонҳои бекенд:
Node.js (JavaScript)
Python (Django, Flask)
PHP
Java
Ruby on Rails

🛠️ Backend бо базаи маълумот (мисли MySQL, MongoDB) кор мекунад.

Сервер


🔹 3. Сервер чист?

Сервер — як компютери махсусест, ки барномаҳои вебро нигоҳ медорад ва ба мизоҷон (users) мефиристад.


🔄 Вақте корбар вебсайтро мекушояд:

Фронтенд HTML/CSS/JS аз сервер ба браузер меравад
Агар маълумот даркор шавад (мисли post ё login), браузер ба сервер дархост мефиристад
Сервер маълумотро аз базаи маълумот гирифта, ба браузер мегузаронад

Full Stack


🔹 4. Full Stack чист?

Full Stack Developer касест, ки ҳам фронтенд ва ҳам бекенд-ро медонад.
